Welcome

Introducing the new CSS Inspector Chrome extension, SureCSS!!! - the perfect tool for web developers and designers who want to easily inspect on any website. With this extension, you can quickly identify any element and highlight it with an outline. Once the extension enabled, you would be able to see the styles as list items in a simple popup window.

Key Features:

The CSS Inspector Chrome extension comes packed with a range of features that make it the perfect tool for developers and designers.
Some of the key features include:

  • Element highlighting: Easily identify any element on a page by simply hovering over it with your mouse.
  • Select styles to inspect: Quickly select the styles in extension popup those to inspect.
  • Element navigation: Move up and down the HTML tree to quickly find the element you're looking for.
  • Identifies most of the elements: Most of the HTML elements are covered for selection.
  • Customizable interface: Selection of styles and choosing them by category gives more flexibility to inspect with quickly.

How to use:

Using the CSS Inspector Chrome extension is easy. Simply install it from the Chrome Web Store, and then click on the extension icon to launch it. From there, you can start inspecting CSS properties on any website.

Steps:

1. Click here to go to Extension Page.

2. Click on Add to Chrome button.

3. Look for new extension badge of SureCSS in browser extensions popup.

4. Pin it to bring badge to top.

5. Start using, have fun playing with it.
